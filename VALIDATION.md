# Validation

Run these commands before publishing:

```bash
npm run validate
npm run scorecard
npm run schema:check
npm run tone:check
npm run test:compile
npm run test
```

Expected result: validation, scorecard, schema check, tone check, compilation, and tests pass.

Real benchmark results are intentionally not invented. Add them under `case-studies/real-*` after running ShipVitals on real projects.
