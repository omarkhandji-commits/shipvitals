# Code of Conduct

ShipVitals contributors must keep technical discussion specific, respectful, and useful.

## Expected Behavior

- Critique evidence and implementation, not people.
- State uncertainty and reproduction limits.
- Protect private project data and credentials.
- Accept correction when a claim is not supported.

## Unacceptable Behavior

Harassment, threats, discrimination, doxxing, deliberate disruption, and publication of private data are not accepted in project spaces.

## Enforcement

Report conduct issues privately through the repository security advisory form. Maintainers may edit or remove content, lock discussions, or restrict participation when needed to protect the community.
