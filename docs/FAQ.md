# FAQ

## Is ShipVitals a checklist?

ShipVitals runs commands, scans source files, applies score caps, and writes JSON and Markdown evidence. The checklist documents release requirements; the runner records whether the project met them.

## Does ShipVitals replace QA?

QA tools and reviewers supply detailed findings. ShipVitals maps those findings to release impact and a verdict.

## Does ShipVitals replace security review?

ShipVitals provides a secret and release-security baseline. Payment, health, legal, financial, enterprise, and sensitive-data projects require specialist review.

## Why use release readiness instead of production readiness?

Production checks cover runtime operation. ShipVitals also covers submission, sale, client delivery, support, privacy, marketplace, and handoff requirements.
