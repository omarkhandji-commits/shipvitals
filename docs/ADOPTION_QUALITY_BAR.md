# Adoption Quality Bar

ShipVitals is ready for public distribution when a new user can complete the checks below without maintainer help.

## Required

- Understand the purpose from the first README paragraph.
- Install from the documented source or registry path.
- Run one audit command.
- Open the generated JSON and Markdown reports.
- Find examples for the project type.
- Reproduce the public benchmark summary.
- Report a bug or false positive through an issue template.
- Distinguish ShipVitals from code review, security review, and browser automation.

## Positioning

ShipVitals checks release evidence and returns a verdict for projects built with coding assistants.
