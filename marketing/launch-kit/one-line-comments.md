# Short Replies

Use these replies only when someone asks about the project.

- ShipVitals audits release evidence and returns a go/no-go verdict for AI-built apps.
- It flags missing runtime proof, fake integrations, skipped tests, mobile evidence gaps, and delivery requirements.
- Tests and specialist reviews supply evidence; ShipVitals applies release policy and records the verdict.
- The report includes blockers, fixes, and retest commands.
- Public projects remain capped until an independent reviewer checks the evidence.
