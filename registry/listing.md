# Marketplace Listing

## Name

ShipVitals

## Short Description

Release evidence and a go/no-go verdict for AI-built apps.

## Long Description

ShipVitals audits a project before publication, sale, submission, deployment, or client delivery.

It records product promise, project commands, runtime proof, visual proof, secret candidates, fake integrations, skipped tests, handoff requirements, marketplace requirements, CI evidence, and unverified areas.

The report returns `READY`, `ALMOST READY`, `NOT READY`, or `DEMO ONLY`, with blockers, fixes, and retest commands.

## Scope
