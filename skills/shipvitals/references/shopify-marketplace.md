# Shopify / Marketplace Gate

- onboarding flow is clear;
- app explains the value in merchant language;
- required scopes are minimal and justified;
- billing/pricing is clear;
- uninstall/data deletion path considered;
- privacy policy and support contact exist;
- app does not depend on fake sample store data for core promise;
- screenshots/listing assets are honest and polished;
- review/submission requirements checked.
