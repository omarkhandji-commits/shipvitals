# Web / SaaS / Landing / MotionVitals Gate

## Visual credibility

- No cheap AI-looking sections.
- No broken spacing, random gradients, inconsistent radii, noisy shadows, or fake dashboards.
- No placeholder copy or vague hero claim.
- Clear CTA above the fold.
- Pricing/contact/support visible when relevant.
- Mobile first viewport must be clean at 390x844.

## Runtime checks

Before READY:

- app starts;
- main route loads;
- key flow executed;
- desktop screenshot captured;
- mobile screenshot captured;
- console/page errors inspected;
- loading/error/empty states are credible;
- links and forms behave correctly.

## Performance baseline

- no obviously huge unoptimized assets;
- no animation jank in key flow;
- no page that feels blocked by unnecessary JS;
- Lighthouse or equivalent recommended for public landing/SaaS.
